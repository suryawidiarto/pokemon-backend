## Run Locally via Docker

Clone the project

```bash
git clone https://github.com/suryawidiarto/pokemon-backend.git
```

Go to the project directory

```bash
cd pokemon-backend
```

Build and run

```bash
docker-compose up --build
```

Server Should be running on

```bash
http://localhost:8080/
```
